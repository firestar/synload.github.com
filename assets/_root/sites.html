AnimeCap

Animecap is a video streaming site. The site is coded in 100% Ruby with MongoDB as the data storage. The videos are pulled automatically from an FTP, and then converted to an h.264 + aac wrapped in the mp4 container. The FTP fetch compares strings to find the latest episodes, and replace deleted videos from the FTP.
[ Ruby on Rails, MongoDB, FFmpeg ]

MCBans

Global banning system for the game, Minecraft. It enables Minecraft server owners to share bans with each other to combat inappropriate players.
[ PHP5, MySQL, Memcached, MongoDB ]
